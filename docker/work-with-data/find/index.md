## Поиск образов в Docker

Образы Docker — это неотъемлемая часть работы с контейнерами. Они содержат все необходимое для запуска вашего приложения. Но что делать, если вам нужно найти конкретный образ среди множества других? 

### Docker Hub и репозитории

Первое место, куда стоит заглянуть, — это Docker Hub. Docker Hub — это центр репозиториев, где вы можете искать и загружать образы. Давайте посмотрим, как это сделать.

#### Поиск образов на Docker Hub

На самом деле, все очень просто. Docker Hub предоставляет веб-интерфейс, где вы можете ввести имя образа в поисковую строку. Например, чтобы найти образ Nginx, просто введите `nginx` в поиск, и вы увидите список доступных образов.

#### Использование командной строки

Также вы можете пользоваться командной строкой для поиска образов. Команда `docker search` — ваш надежный помощник в этом деле. Давайте разберемся, как именно:

```bash
docker search nginx
```

Эта команда ищет образы, связанные с Nginx, и возвращает список с информацией о популярных репозиториях. Как видите, результат поиска показывает имя образа, описание и количество звезд (указывающих на популярность образа).

## Поиск контейнеров в Docker

Когда у вас запущено множество контейнеров, важно быстро находить необходимые. Docker предлагает эффективные способы для управления и поиска контейнеров.

### Команда `docker ps`

Наиболее распространенный способ поиска контейнеров — это использование команды `docker ps`. Она показывает список всех запущенных контейнеров.

```bash
docker ps
```

Вы можете увидеть ID контейнера, имя образа, команду, статус и другую полезную информацию. Если вы хотите увидеть все контейнеры, включая остановленные, используйте флаг `-a`:

```bash
docker ps -a
```

### Фильтрация контейнеров

Иногда нужно отсортировать контейнеры по определенным критериям, например, по имени или статусу. Docker позволяет это сделать с помощью флагов фильтрации:

```bash
docker ps -f "name=my_container"
```

В этом примере мы фильтруем контейнеры по имени `my_container`. Аналогично, вы можете фильтровать по статусу, используя `status`:

```bash
docker ps -f "status=running"
```
